They press 2 on their keypad. The AI transfers the call to your team's phone — could be a single number or a round-robin to multiple staff. The transfer takes ~3 seconds.

Quick start

Add a custom trigger that fires whenever an order moves to "processing" status:

add_action( 'woocommerce_order_status_processing',
  function( $order_id ) {
    $order = wc_get_order( $order_id );
    voicenimble_trigger_call(
      'agent_daraz_order',
      $order->get_billing_phone(),
      [
        'customer_name'  => $order->get_billing_first_name(),
        'order_id'       => $order_id,
        'order_total'    => $order->get_total(),
      ]
    );
  }
);
